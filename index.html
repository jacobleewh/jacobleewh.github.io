BridgeGen

Intergenerational Community Platform

BridgeGen is a full-stack Python/Flask web application built to bridge the generational gap, creating a safe and vibrant space where seniors and youths connect through shared stories, local events, and real-time conversation.

I led the development of the real-time chat system using Flask-SocketIO, enabling private and group messaging with multimedia support. The backend follows a clean MVC architecture with SQLAlchemy for robust database management, while CRUD principles drive efficient data persistence across all user content. This modular design allowed the team to simultaneously build out features including a gamified rewards system and an AI chatbot ‚Äî all within a scalable, conflict-free codebase.
